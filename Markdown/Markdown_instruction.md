## Списки

Чтобы добавить ненумерованные списки, необходимо пунктвы выделить звёздочкой (*), например:
* Элемент 1
* Элемент 2
* Элемент 3

Чтобы доавить нумерованные списки, достаточно просто пронумеровать их:

1. Пункт первый
2. Пункт второй
